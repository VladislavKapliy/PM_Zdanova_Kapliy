## Мета фаза

Фаза 2 посвящена планированию выполнения Фазы 3 через оптимальное распределение задач между членами команды с учётом предпочтений, навыков и баланса нагрузки.

# В ходе выполнения работы:

### 1. Декомпозиция Фазы 3
1. Определили 8 ключевых задач Фазы 3
2. Для каждой задачи прописали:
- Описание работы
- Ожидаемый результат
- Критерии завершённости
3. Создали Work Breakdown Structure (WBS)

### 2. Оценка предпочтений команды:
1 - не хочу/не могу;
3 - нейтрально/могу сделать;
5 - очень хочу.

### 3. Математическая оптимизация распределения
- Сформулирована задача максимизации суммарного предпочтения
- Учтены ограничения по балансу нагрузки
- Решение найдено с помощью OpenSolver

### 4. Создание матрицы ответственности
- Разработана матрица RACI с чёткими ролями
- Определены Responsible, Accountable, Consulted, Informed

### 5. Разработка плана
- Составлен детальный план на 10 рабочих дней
- Определены контрольные точки и митинги
